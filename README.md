# alfa_investing
Скрипт для сбора данных с Investing.com + консольное приложение Currency.py

Файл get_currency.py обращается к сайту https://ru.investing.com/charts/forex-charts, получает данные и записывает в базу данных. Скрипт настроен на бесконечный цикл. Выход принудительным прерыванием с помощью Ctrl+C.

В файле get_max_min_currency.py написана функциях, которая на основе полученных данных строит запрос, который будет выводить максимальную и минимальную цену выбранной валюты.

Файл currency.py - консольное приложение, которое работает в двух режимах:
1. Выводит максимальную и минимальную цену выбранной валюты за выбранной период
2. Выводит количество записей и сами записи за выбранную дату для выбранной валюты с опциональным аргументом limit, который ограничивает количество записей

Примеры работы консольного приложения:
 
# Рассчет максимальной и минимальной цены
./currency.py minmax usdrub 2022-06-01 2022-06-06
MIN = 75.75
MAX = 76.76

# Количетво записей и сами записи
./currency.py list usdrub 2022-06-01 2022-06-06 --limit=3
Count = 3
1. 75.75
2. 75.70
3. 75.65
